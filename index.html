How to Update This Page

  1. Copy your finished leaderboard from Excel into a Google Sheet tab.
  2. Make sure the sheet columns are: Rank, Player, Points, Weeks Played.
  3. Publish that specific sheet tab to the web as CSV.
  4. Replace the sheetCsvUrl value in this file with your published CSV link.
  5. Save the file and upload it to GitHub Pages.

Tip: Keep row 1 as your header row in Google Sheets.
